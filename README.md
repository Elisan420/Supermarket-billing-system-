# Supermarket Billing System (Python)

A simple command-line supermarket billing system built using Python.  
It allows a cashier to select items, enter quantities, and generates a printed receipt with date, time, and total cost.

##  Features
- Item selection menu  
- Quantity input  
- Automatic bill calculation  
- Receipt printing with timestamp  
- Cashier name display  

## Tech Stack
- Python 3  
- datetime module
